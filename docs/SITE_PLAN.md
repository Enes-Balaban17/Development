# Site Plan

This document defines the first version of the portfolio website before coding starts.

## Main Sections

1. Header / Navigation
2. Hero
3. About Me
4. Notes
5. Skills
6. Projects
7. Email Newsletter
8. Contact
9. Footer

## Header / Navigation

Planned navigation items:

- Notes
- Skills
- Projects
- About
- Contact
- Color
- Theme

## Hero

The hero section should introduce the website quickly.

Planned content:

- Name
- Short developer title
- Short introduction paragraph
- Links to Projects, Email, GitHub, LinkedIn, and CV

## About Me

Education and personal background will be included inside the About section instead of creating a separate Education section.

Planned topics:

- Computer Programming background
- Interest in software development
- Interest in data analysis and artificial intelligence
- Interest in hardware-based and drone-related projects

## Notes

The website will not include a traditional blog. Instead, it will have a Notes section.

Possible note types:

- Learning logs
- Development notes
- Project planning notes
- Technical reminders
- GitHub portfolio progress notes

## Skills

The Skills section will be presented visually with icons or icon-like cards.

Planned categories:

- Programming Languages
- Web Development
- Database Systems
- Tools
- Artificial Intelligence & Data Analysis
- Hardware & Drone Projects

## Projects

Projects will be listed in a clean timeline/list format.

Each project item should include:

- Year
- Project name
- Short description
- Technologies
- Source link
- Demo or Details link when available

## Email Newsletter

The website should include an email newsletter style section.

Initial version:

- Newsletter form as a visual component
- Coming soon note
- Active email contact button

Future version:

- Connect to a newsletter service such as Buttondown, Substack, or Mailchimp

## Contact

Planned links:

- Email
- GitHub
- LinkedIn
- CV

## Design Direction

- Minimal layout
- Readable typography
- Dark and light theme support
- Accent color selection
- Clean spacing
- Responsive design
- Simple project and note lists
