# Portfolio Website

A minimalist personal portfolio website inspired by clean developer websites. This project is built to showcase my projects, notes, skills, and development journey in a simple and accessible way.

## Project Purpose

The main purpose of this website is to create a personal space where I can present my software projects, technical notes, skills, contact links, and portfolio progress.

This website will not include a traditional blog section. Instead, it will include a **Notes** section for short technical notes, learning logs, and project-related reminders.

## Planned Sections

- Home / Hero
- About Me
- Notes
- Skills
- Projects
- Email Newsletter
- Contact
- Footer

## Design Direction

The website will follow a clean and minimal developer portfolio style with:

- Simple typography
- Dark and light theme support
- Accent color selection
- Minimal animations
- Notes section instead of a blog
- Skills section with visual icons
- Project list with source and demo links
- Email newsletter style area
